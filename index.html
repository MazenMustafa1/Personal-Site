Projects

These Projects have been built using Flutter and Java FX

Quizzem – Quiz Creation And Management Application (Still in development)

Description:

Quizzem is a mobile application designed for teachers to create and manage quizzes for their students. It allows teachers to create quizzes and set a time limit for each quiz. After quiz creation, the application generates a unique code for the quiz, which can be shared with students.


Shop App - E-commerce Application with Provider State Management

Description:

Shop App is a multi-screen mobile application that allows users to browse, add, delete, and edit products in a virtual store. The application uses Firebase Realtime Database for data management and implements Provider state management for efficient data flow between different screens. Authenticated users can perform CRUD (Create, Read, Update, Delete) operations on products, and the changes are reflected in real time for other users.


Great Places App – Photo Capture and Location Tracking Application

Description:

Great Places is a mobile application that allows users to capture photos of interesting places using their mobile camera, save the location of those places using Google Maps, and store them locally using the Sqflite package. The application allows users to manage and view their collection of great places for future reference.


Shortest Job First Project – An Operating Systems’ algorithm for handling processes

Description:

I developed a JavaFX GUI application that simulates the Shortest Job First (SJF) algorithm used in operating systems to handle processes. The program allows users to input the number of processes, arrival time, and burst time for each process. It then generates a Gantt chart and calculates the response time, waiting time, and turn-around time for each process, as well as the averages for all processes. This project allowed me to deepen my understanding of operating systems and algorithms, as well as to improve my Java programming skills.
